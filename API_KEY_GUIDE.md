# Google AI Studio API Key - Anleitung

## So erhalten Sie einen gültigen API Key:

### Schritt 1: Zur Website gehen

Öffnen Sie in Ihrem Browser:

```
https://aistudio.google.com/apikey
```

### Schritt 2: Einloggen

Melden Sie sich mit Ihrem Google-Konto an.

### Schritt 3: API Key erstellen

1. Klicken Sie auf **"Create API Key"** oder **"Get API Key"**
2. Wählen Sie ein Google Cloud Projekt (oder erstellen Sie ein neues)
3. Der Key wird automatisch generiert

### Schritt 4: Key kopieren

- **WICHTIG**: Kopieren Sie den Key **komplett**
- Achten Sie darauf, dass **keine Leerzeichen** am Anfang oder Ende sind
- Der Key sieht ungefähr so aus: `AIzaSyC...` (39 Zeichen)

### Schritt 5: In die App einfügen

1. Öffnen Sie das GEM-Apex Programm
2. Fügen Sie den Key in das Feld "Google API Key" ein
3. Klicken Sie auf "Connect AI"

## Häufige Fehler:

❌ **"API_KEY_INVALID"**

- Der Key ist falsch oder wurde gelöscht
- Erstellen Sie einen neuen Key

❌ **Leerzeichen im Key**

- Kopieren Sie den Key nochmal sorgfältig
- Die neue Version entfernt automatisch Leerzeichen

❌ **Key zu kurz**

- Google API Keys haben ~39 Zeichen
- Überprüfen Sie, ob Sie den ganzen Key kopiert haben
